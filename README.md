# Description
The Admin Panel is a web-based management dashboard built to control and monitor every core activity of the Doctor Appointment System.

# Objectives
Its main objectives are:

To allow administrators to manage doctors, including adding, updating, verifying, or removing profiles

To monitor patient accounts and ensure proper onboarding

To handle appointments, including viewing, modifying, or canceling any booking

To maintain specializations, clinic data, and scheduling details

To display analytics such as total doctors, patients, appointments, cancellations, growth trends, etc.

To enable secure login with admin-only privileges

To ensure smooth operation and regulation of the doctor–patient appointment ecosystem


# Tools & Technologies Used:
Frontend:

React.js → for building the component-based admin interface

HTML5 → markup structure

TailwindCSS → modern and utility-first styling

JavaScript (ES6+) → logic & state management
